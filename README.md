quantitative research and trading.

- **Statistical Analysis** — return distributions, volatility modeling, risk metrics
- **Financial Modeling** — pricing models, portfolio construction, backtesting
- **Software Engineering** — clean code, version control, reproducible research

---

## Projects

### 1. Stock Returns Analyzer
Pulls historical price data for any ticker and computes key risk/return metrics.

**Concepts covered:**
- Daily and cumulative return calculation
- Annualized return and volatility
- Sharpe ratio
- Maximum drawdown

**Libraries:** yfinance, pandas, matplotlib
